## JSP게시판 개인프로젝트
***
    oracle DB와 eclipse를 연동하여 JSP게시판을 만들었습니다.
    주요 기능은 게시글 목록, 글쓰기, 상세보기, 파일 업로드 및 다운로드, 수정, 삭제입니다.
#### Frontend
    HTML5, CSS3, JavaScript, JQuery, bootstrap
#### Backend
    Java, JSP
#### DB
    Oracle db
#### WAS
    Apache Tomcat
